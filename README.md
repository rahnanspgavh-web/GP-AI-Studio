# GP-AI-Studio

GP-AI-Studio is a modular AI automation platform designed to help creators and teams manage YouTube workflows, websites, content generation, and productivity tools.

## Overview

- Modular architecture with dedicated packages for AI, automation, health, spirituality, temple, relaxation, food, YouTube, website, and shared resources.
- Supports structured project organization for prompts, assets, data, outputs, logs, documentation, and scripts.
- Designed for future expansion with clean separation between core application logic and domain modules.

## Project Structure

- `app/` - primary application package
- `modules/` - domain-specific modules and shared utilities
- `prompts/` - prompt templates and prompt engineering assets
- `assets/` - static files, media, and resource assets
- `data/` - datasets and data resources
- `output/` - generated content and export files
- `logs/` - application logs and diagnostics
- `tests/` - unit tests and integration tests
- `docs/` - documentation and technical references
- `website/` - website assets and deployment files
- `config/` - configuration and environment settings
- `scripts/` - helper scripts and automation tools

## Getting Started

1. Create a Python virtual environment.
2. Install dependencies from `requirements.txt`.
3. Add project logic inside `app/` and the `modules/` package.
